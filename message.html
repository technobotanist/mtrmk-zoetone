Zoetone is an interactive 3D multi-channel audio visualizer and performance tool by MTR Magic Key.

  • Assign, control, and mix up to 6 independent audio channels in real time.
  • Each channel can be loaded with your own audio URL or use the demo tracks.
  • Live audio analysis powers responsive 3D animations and visual effects for each channel.
  • Dynamic UI lets you add/remove channels, change tracks, and control playback.
  • Built with A-Frame and WebXR for immersive, portfolio-ready demos.

Explore, remix, and visualize music in a spatial, interactive environment.
